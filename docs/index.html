What is XFlow?

XFlow is an ambitious attempt to reimagine the modern hardware and software stack.

What does this New Stack Look Like?

At the lowest level of XFlow's stack will be an FPGA bytestream generator.

Essentially this allows a digital circuit to be created from scratch.

Above that will be a language xflow-lang and a database xflow-qengine. This language will allow the database to be queried and will be built on tech from Timely Dataflow that allows ultra-efficient incremental queries.

Above this data layer will be a UI layer as well as a Network layer.
